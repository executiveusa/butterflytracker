# Cinematic Components Audit
Inspected `robonuggets/cinematic-site-components` on 2026-06-23. Useful patterns: sticky stack, zoom parallax, split scroll, cursor reactive, spotlight border, curtain reveal, text mask. Rejected: glitch effect, mesh gradients, cyberpunk-like color shift, particle button, dynamic island, dock nav because they conflict with nature-documentary direction or add startup/gaming cues. Adapted ideas, not copied wholesale, into `src/components/cinematic/*`: atmospheric mist, butterfly trail, cinematic hero, chapter reveals, CTA, parallax/sunbeam/transition helpers. Dependency impact: none; existing React/Tailwind/R3F stack is enough. Accessibility: semantic sections, focusable butterfly controls, reduced-motion CSS. Performance: CSS layers and lazy Canvas only; no large assets.
